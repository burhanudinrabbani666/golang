# Multiple Main function

- Di golang function dalam module/ project adalah unik, artinya kita tidka boleh membuat nam function yang sama
- Oleh karena itu, jika embuat fle baru, misal `sample.go`, lau membuat nama function yang sama yaitu **main**
- Maka kita tidak bisa melakukan **build** module, karena **main** function tersebut duplikat dengan yang ada di **main** function `helloworld.go`
