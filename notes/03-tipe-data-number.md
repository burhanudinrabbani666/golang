# Tipe data Number

Ada dua jenis tpe data Number, yaitu:

1. Integer
2. Floating Point

Tipe Data integer (1)

| Tipe data | Min                  | Max                 |
| --------- | -------------------- | ------------------- |
| int8      | -128                 | 128                 |
| int6      | -32768               | 32768               |
| int32     | -2147483648          | 2147483648          |
| int64     | -9223372036854775808 | 9223372036854775808 |

Tipe Data integer (2)

| Tipe data | Min | Max                  |
| --------- | --- | -------------------- |
| uint8     | 0   | 255                  |
| uint6     | 0   | 65536                |
| uint32    | 0   | 4294967295           |
| uint64    | 0   | 18446744073709551615 |

Tipe data Floating Point

| Tipe data  | Min                                                  | Max   |
| ---------- | ---------------------------------------------------- | ----- |
| float32    | 1.18x10^38                                           | 255   |
| float64    | 2.23x10^-308                                         | 65536 |
| complex64  | complex number with float32 real and imaginary parts |
| complex128 | complex number with float64 real and imaginary parts |

Alias

| Tipe data | Alias         |
| --------- | ------------- |
| byte      | uint8         |
| rune      | int32         |
| int       | Minimal int32 |
| uint      | Minimal int32 |
