# Tipe data Boolean

Tipe data boolean adalah tipe data yang memiliki nilai dua nilai, yaitu `true` dan `false`
Di Go-lang, tipe data boolean direpresentasikan menggunakan kata kunci `bool`

```go
package main

import "fmt"

func main(){
	fmt.Println("Benar = ", true)
	fmt.Println("Salah = ", false)
}
```
